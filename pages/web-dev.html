Курс: Веб-Разработка (PHP,JS)


Возраст: 14-17 лет
Требования: обязательно изучение языка С# или СИ (с отличием). Обязательно изучение HTML и CSS
Описание:
Любой современный веб-ресурс обладает бэкендом, который обеспечивает его работу. С помощью HTML и CSS можно создавать простые сайты, но для создания порталов, поисковых систем, новостных ресурсов и прочих сложных систем, только версткой не обойтись. В рамках курса изучаются два языка – JavaScript и PHP, а также основы работы с серверами и хостингами сайтов. JavaScript – язык, который в данное время держит лидерство как самый популярный инструмент разработки. С его помощью можно создавать веб-системы, динамический контент, галереи изображений и прочее. PHP – настоящий мастодонт мира IT, который лежит в основе всех современных веб-движков, а также обладает мощным функционалом в работе с базами данных. Итогом изучения курса станет веб-портал с готовым дизайном и функционалом, определенным в техническом задании выданном преподавателем.
